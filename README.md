# Professional Portfolio

A clean, professional static portfolio website scaffold created to showcase your resume and experience.

## Files

- `index.html` — main portfolio page.
- `styles.css` — responsive styling and layout.
- `script.js` — mobile navigation toggle.

## Customize

1. Open `index.html`.
2. Replace placeholder text like `Your Name`, `[Job Title]`, `[Current Company]`, and project descriptions with your resume details.
3. Update contact info and links.

## Run locally

1. Install dependencies with `npm install`
2. Start the development server with `npm run dev`

## Build

- `npm run build`
- `npm run preview`

## Deploy

This portfolio is built with Vite and is ready for Vercel deployment. Use `npm run build` for production builds and connect the repository to Vercel for automatic deployment.
